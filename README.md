# 💰 Telegram Investment Bot

A Telegram bot for investment platform with proof channel, support, and investment plans.

## ✨ Features

- ✅ Welcome message with user's name
- 📢 Proof Channel button
- 📞 Contact Support button  
- 📊 Investment Plans (Basic, Silver, Gold, Platinum)
- ❓ Help/FAQ section
- 🔙 Back to Main Menu functionality

## 🚀 Setup Instructions

### 1. Get Bot Token
- Open Telegram
- Search for `@BotFather`
- Send `/newbot`
- Follow instructions and get token
